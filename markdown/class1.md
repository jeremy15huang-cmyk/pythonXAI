# 🐍 Python 筆記

## 1. `print()`

### 🌟 它是做什麼的？

`print()` 就像是在跟電腦說：

> 「請把這句話顯示在螢幕上！」

### 💻 範例

```python
print("Hello")
```

### 🖥️ 執行結果

```
Hello
```

### 📝 小提醒

- 文字要放在雙引號 `"` 或單引號 `'` 裡面。
- 少了括號或引號，程式就會出錯。

---

## 2. `input()`

### 🌟 它是做什麼的？

讓電腦問使用者問題，等使用者輸入答案。

```python
name = input("請輸入你的名字：")
print("你好", name)
```

---

## 🎯 今天重點

✅ `print()`：把東西顯示出來。
✅ `input()`：讓使用者輸入資料。
✅ `=`：把資料放進變數裡保存。

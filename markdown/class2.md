2.  Python 的 for 迴圈

## 🎯 什麼是 for 迴圈？

有時候，我們希望電腦一直重複做同一件事情，例如：

- 印出 5 次「Hello」
- 數數 0 到 4
- 重複畫出相同的圖形

如果每次都自己寫一次程式，會很麻煩，所以我們可以使用 **`for` 迴圈**。

`for` 的功能就是：

> **讓電腦按照順序，把同一件事情重複做好幾次。**

---

## 🎯 for、in、range() 一起使用

`for` 通常會搭配 **`in`** 和 **`range()`** 一起使用。

基本寫法：

```python
for i in range(5):
    print(i)
```

這一行程式可以分成三個部分：

### ① `for`

表示開始進行重複工作。

### ② `in`

表示從一串資料裡，一個一個拿出資料。

### ③ `range(5)`

`range()` 會產生一串連續的數字。

```python
range(5)
```

會得到：

```text
0、1、2、3、4
```

⚠️ 注意：

`range(5)` **不會包含 5**。

也就是說：

| 程式        | 產生的數字    |
| ----------- | ------------- |
| `range(3)`  | 0、1、2       |
| `range(5)`  | 0、1、2、3、4 |
| `range(10)` | 0～9          |

可以記住一句話：

> **`range(數字)` 會從 0 開始，到「數字前一個」結束。**

---

## 🎯 什麼是迴圈變數？

程式中的：

```python
for i in range(5):
```

`i` 稱為**迴圈變數**。

它就像一個小盒子，每一次都會放進不同的數字。

| 第幾次 | i 的值 |
| ------ | ------ |
| 第一次 | 0      |
| 第二次 | 1      |
| 第三次 | 2      |
| 第四次 | 3      |
| 第五次 | 4      |

每跑一次迴圈，`i` 都會自動換成下一個數字。

其實它不一定要叫 **`i`**，也可以自己取名字，例如：

```python
for number in range(5):
    print(number)
```

或

```python
for student in range(5):
    print(student)
```

只要名稱有意義、自己看得懂即可。

---

## 🎯 程式執行過程

程式：

```python
for i in range(5):
    print(i)
```

電腦會依照下面的順序執行：

第一次：

```text
i = 0
印出 0
```

第二次：

```text
i = 1
印出 1
```

第三次：

```text
i = 2
印出 2
```

第四次：

```text
i = 3
印出 3
```

第五次：

```text
i = 4
印出 4
```

最後畫面會看到：

```text
0
1
2
3
4
```

## 💡 生活中的例子

想像老師說：

> 「請第一位到第五位同學依序站起來。」

老師會按照順序叫：

1️⃣ 第一位同學
2️⃣ 第二位同學
3️⃣ 第三位同學
4️⃣ 第四位同學
5️⃣ 第五位同學

`for` 迴圈也是一樣，它會按照順序，一次拿出一個數字來使用。

---

## 📝 本章重點

✅ `for`：讓程式重複執行。

✅ `in`：從一串資料中，一個一個取出資料。

✅ `range(5)`：產生 **0、1、2、3、4**，不包含 5。

✅ `i`：迴圈變數，每次都會拿到一個新的數字。

✅ `print(i)`：把目前拿到的數字顯示在畫面上。

---

## 🎯 一句話記住

> **`for` 迴圈就像老師依照座號點名，電腦會依序拿出每個數字，並重複完成同一件事情。**
